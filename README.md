# ☁️ Cloud Infrastructure Automation Lab

## 📌 Overview

This project simulates a cloud environment using Infrastructure as Code (IaC) and configuration management.

It demonstrates how to provision infrastructure and configure servers automatically using Terraform and Ansible.

---

## 🎯 Objectives

- Provision infrastructure using Terraform
- Configure instances using Ansible
- Simulate a DevOps workflow
- Apply Infrastructure as Code principles

---

## 🏗️ Architecture

- Terraform for infrastructure provisioning
- Ansible for configuration management
- Modular and scalable design

---

## ⚙️ Features

- Infrastructure provisioning (VM simulation)
- Automated server configuration
- SSH setup and hardening
- DevOps workflow simulation

---

## 🚀 How It Works

1. Terraform provisions infrastructure
2. Ansible configures the servers
3. Environment becomes ready automatically

---

## 🧠 Skills Demonstrated

- Infrastructure as Code (Terraform)
- Configuration management (Ansible)
- Cloud concepts
- DevOps practices
- Automation pipelines

---

## ⚠️ Notes

This project simulates cloud environments and does not require a real cloud account.
